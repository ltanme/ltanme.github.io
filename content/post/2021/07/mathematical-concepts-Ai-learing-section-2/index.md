# AI学习--数学基础概念



#### 绝对值 

绝对值是指一个数在数轴上所对应点到原点的距离，例如，3的绝对值为3，-3的绝对值也为3。数字的绝对值可以被认为是与零的距离。

> 应用举例：

```py
from keras import backend as K
# b是生成一个4x4 二维 -1 ~ 0 的随机数
b = K.random_uniform_variable(shape=(4, 4), low=0, high=-1) # 均匀分布
# 原始打印
print(b)
# 取绝对值后打印
print(K.abs(b))
```
